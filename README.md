# Data Structures and Algorithms in C#

A collection of common data structures and algorithms implemented in C#, focusing on clean code and efficient implementations.

## Project Structure

```
algo_dsa/
├── Array/
│   └── Anagram.cs          # Anagram checking implementations
├── BinarySearchTree/
│   ├── BinarySearchTree.cs # BST operations
│   └── TreeNode.cs        # Tree node structure
└── LinkedList/
    ├── LinkedList.cs      # Linked list operations
    └── Node.cs           # List node structure
```

## How to Run

1. **Prerequisites**
   - .NET 6.0 or later
   - Visual Studio Code or Visual Studio 2022

2. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/algo-dsa.git
   cd algo-dsa
   ```

3. **Build the Project**
   ```bash
   dotnet build
   ```

4. **Run the Project**
   ```bash
   dotnet run --project algo_dsa
   ```

5. **Run Tests**
   ```bash
   dotnet test
   ```
